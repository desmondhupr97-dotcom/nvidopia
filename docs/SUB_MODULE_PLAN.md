# SUB_MODULE_PLAN: docs

## 模块目标与边界

项目文档中心。集中管理架构文档、开发运行手册、验收文档与变更日志，为团队协作和新成员 onboarding 提供参考。

**边界**：
- 负责：架构设计文档、本地开发指南、API 使用手册、验收报告模板、变更日志。
- 不负责：代码实现、契约定义（在 contracts/ 中）。
